# 📱 Calculadora de IMC - Jetpack Compose

Uma aplicação Android moderna e intuitiva para cálculo de Índice de Massa Corporal (IMC), desenvolvida inteiramente em **Kotlin** com a biblioteca de UI declarativa **Jetpack Compose**.

---

## 🚀 Tecnologias Utilizadas

* **Kotlin**: Linguagem principal de desenvolvimento.
* **Jetpack Compose**: Construção de interface moderna e reativa.
* **Material Design 3**: Componentes visuais atualizados.
* **State Management**: Gerenciamento de estado com `remember` e `mutableStateOf`.
* **Custom Modifiers**: Estilização avançada de componentes.

## ✨ Funcionalidades

* Cálculo preciso de IMC (Índice de Massa Corporal).
* Classificação dinâmica do resultado (Abaixo do peso, Peso ideal, Sobrepeso, etc.).
* Interface responsiva com validação de campos numéricos.
* Design limpo e focado na experiência do usuário (UX).

## 🏗️ Como Rodar o Projeto

1.  Clone este repositório:
    ```bash
    git clone [https://github.com/seu-usuario/calculadora-imc-compose.git](https://github.com/seu-usuario/calculadora-imc-compose.git)
    ```
2.  Abra o **Android Studio** (versão Ladybug ou superior recomendada).
3.  Importe o projeto e aguarde a sincronização do **Gradle**.
4.  Execute em um emulador ou dispositivo físico com **Android 8.0+**.
